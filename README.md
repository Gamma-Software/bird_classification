# Bird Classification
Make use of my camera ip pointing toward a bird feeder and a jetson nano to detect a bird and classify it

## Use
Rename credential_test.txt to credential.txt and add your email credential\
I tested with gmail:
* the username is you gmail mail address
* the password is a custom password generated via the Gmail account (follow the tuto https://umd.service-now.com/itsupport?id=kb_article&article=KB0015112&sys_kb_id=76433076dbdf8c904cb035623996194b&spa=1)
